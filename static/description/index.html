Key Features:

  • Can enable/disable 2FA from backed for specific user.
  • Can send 2FA QR code to user via Email.
  • Easy to use. No extra configuration required.

Secure login into your Odoo account.

With our Two Factor Authentication app you will need to enter a time-based passcode when logging into odoo account. This passcode changes every 30 seconds, so even if someone knows your password, they will not be able to login to your account.

Why to get secured with Two Factor Authentication?

As every time anyone enters your username and password (in case) to login, instantly a new page pop up and ask for a verification code. This unique Time-based One-Time Password will be generated only on your cellphone, using the Google Authenticator App. So, this new layer will add up to the strength of the security at your end. Any app that supports the Time-based One-Time Password (TOTP) algorithm can be used to generate the security token for you.

External python library 'qrcode' is required.

To install it in odoo 9,10 use command: sudo pip install qrcode

To install it in odoo 11,12 use command: sudo pip3 install qrcode
